Object methods

A function that is a property of an object is called its method.

A function Expression to create a function and assign it to the property user.sayHi of the object.

let user = {
name: "John",
age: 30
};

user.sayHi = function() {
alert("Hello!");
};
user.sayHi(); // Hello!

a pre-declared function as a method,

let user = {
// ...
};

// first, declare
function sayHi() {
alert("Hello!");
}

// then add as a method
user.sayHi = sayHi;

user.sayHi(); // Hello!

“this” in methods

an object method needs to access the information stored in the object to do its job.

To access the object, a method can use the this keyword.

example:
let user = {
name: "John",
age: 30,

sayHi() {
// "this" is the "current object"
alert(this.name);
}

};

user.sayHi(); // John

  • we can omit "function" and just write sayHi().
  • The value of this is the object “before dot”, the one used to call the method.
  • Here during the execution of user.sayHi(), the value of this will be user.

“this” is not bound

In JavaScript, keyword this can be used in any function, even if it's not a method of an object.

In JavaScript this is “free”, its value is evaluated at call-time and does not depend on where the method was declared, but rather on

what object is “before the dot”.
