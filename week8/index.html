Transforms

transform property

translate, rotate, scale, and/or skew any element on the page.

manipulate an element's appearance using transform functions

Translation functions

move elements left, right, up, or down.

translation function moves elements without impacting the flow of the document.

The translate(x,y) function moves an element x from the left, and y from the top

transform: translate(45px, -45px);

Note: Transforms and Older Browsers

Transforms require vendor prefixing for IE9, Android up to 4.4.3, iOS8, and Blackberry 10.
To make the aforementioned code work in IE9 and older mobile WebKit browsers, you would include the following:

-webkit-transform: translate(45px,-45px); /* iOS8, Android 4.4.3, BB10 */
-ms-transform: translate(45px,-45px); /* IE9 only */ transform: translate(45px,-45px);

If you only want to move an element vertically or horizontally, you can
use the translatex or translatey functions respectively.

move 45px to the right along the x axis,

transform: translateX(45px);

move up along the y axis by 30px

transform: translateY(-30px);

Scaling

scale(x,y) function

horizontally then vertically

If only one value is provided, it will be used for both the x and y values

scale(1) would leave the element the same size

scale(2) would double its proportions

Providing different values will distort the element

scaleX(x) or scaleY(y) functions. These functions will scale only the horizontal dimensions or only the vertical dimensions respectively.s

Rotation

rotate() function

rotates an element around the point of origin by a specified angle value.

angles are declared in degrees, with positive degrees moving clockwise and negative moving counterclockwise. In addition to degrees, values can be provided in grads, radians, or turns

remember that transforms are applied in the order provided

A Skewed Perspective

Changing the Origin of the Transform

control the origin from which your transforms are applied

transform-origin property.

Rotating an ellipse is more noticeable if the transform-origin is set to something other than the default 50% 50%

Note: Choose Your Ordering Carefully

if you rotate before translating, your translate direction will be on the rotated axis.

Support for Internet Explorer 8 and Earlier

While CSS3 transforms are unsupported in IE before version 9, you can mimic these effects with other CSS properties, including filters. To mimic translation use position: relative;, and top and left values

You can also scale an element by altering its width and height, or changing the font-size

altering a width, height or font-size alters the space allocated for the element and can affect the layout.

Transitions

Las transiciones permiten que los valores de las propiedades de CSS cambien con el tiempo, proporcionando esencialmente animaciones simples.

con la propiedad de transición CSS, la transición de color puede ser gradual. Si el navegador no admite transiciones, el cambio será inmediato en lugar de gradual, lo cual está bien y es accesible.

Las transiciones CSS se declaran junto con los estilos regulares en un elemento.

Often the change will be due to different styles applied to a hover state; however, transitions will work equally well if the property in question is changed by adding a class, or otherwise using JavaScript to change state.

steps to create a simple transition using only CSS:

  1. Declare the original state of the element in the default style
    declaration.
  2. Declare the final state of your transitioned element; for
    example, a :hover state.
  3. Include the transition functions in your default
    style declaration     using the transition properties, including:
    transition-property, transition-duration,transition-timing-function, and transition-delay.

The important point to note is that the transition is declared in the default or originating state.
